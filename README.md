# EEG Data Science Project

Proyecto de análisis y preprocesamiento de EEG utilizando MNE-Python.

## Estructura

- data/raw → Datos crudos (no incluidos en el repositorio)
- data/processed → Datos procesados
- notebooks → Análisis y preprocesamiento
- src → Funciones reutilizables

## Fases completadas

✔ Inspección de datos  
✔ Limpieza de señal  
✔ Filtrado y eliminación de ruido
